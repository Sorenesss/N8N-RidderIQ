# N8N RidderIQ Node

Custom node for n8n to interact with the RidderIQ API.

## Installation

1. **Clone the repository** (replace `<token>` with your GitHub personal access token):

```bash
git clone https://<token>@github.com/Sorenesss/N8N-RidderIQ.git
```

2. **Enter the project folder**:

```bash
cd N8N-RidderIQ
```

3. **Run the deployment script**:

```bash
chmod +x deploy-node.sh
./deploy-node.sh
```

4. **Clean up (optional)**:

```bash
cd ..
rm -rf N8N-RidderIQ
```

---

After this, your RidderIQ node should be installed in n8n and ready to use.
